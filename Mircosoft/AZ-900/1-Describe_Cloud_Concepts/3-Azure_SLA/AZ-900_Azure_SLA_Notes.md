# Azure SLA (Service Level Agreement) -- AZ-900 Notes

## What is an SLA?

SLA (Service Level Agreement) is a formal guarantee from Microsoft Azure
defining uptime, performance, and compensation.

## Availability

Availability is expressed as uptime percentage per month. Higher SLA
means less downtime.

## Key Points

-   SLA = Uptime guarantee
-   Measured monthly
-   Violations → Service credits
-   Free services → Usually no SLA
-   Architecture affects availability

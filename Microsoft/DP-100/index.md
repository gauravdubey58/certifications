## 🎯 Candidate Profile

You should have subject matter expertise in:
- Applying data science and machine learning on Azure
- Azure Machine Learning (AML) workspace, compute, and SDK v2
- MLflow for experiment tracking and model registration
- Azure AI Foundry for language model optimization
- RAG, prompt engineering, and fine-tuning
- At least **1 year** experience implementing end-to-end ML solutions on Azure

## ⚡ Quick Revision — Key Facts

| Fact | Value |
|------|-------|
| Pass score | 700 / 1000 |
| Two equal largest domains | Train & Deploy + Language Models (25–30% each) |
| Primary ML platform | Azure Machine Learning (AML) |
| Experiment tracking tool | MLflow |
| Language model platform | Azure AI Foundry |
| AutoML supported task types | Classification, Regression, Forecasting, CV, NLP |
| Online endpoint types | Managed (serverless) and Kubernetes |
| Batch endpoint use | Asynchronous, large-scale, offline scoring |
| Compute for training | Compute Cluster (AmlCompute) |
| Compute for dev/notebooks | Compute Instance |
| Hyperparameter tuning service | Azure ML Sweep Job |
| Sampling methods | Grid, Random, Bayesian, Sobol |
| RAG vector store options | AI Search, FAISS, Chroma, Azure Cosmos DB |
| Fine-tuning base model source | Azure AI Foundry Model Catalog |
